# Chat
- Web-App (Messenger)
# Dependencies
- express - using for processing HTTP-requests and routes.
- socket.io - helping create chat with two-side connection against client and server.
- cors - helping to manage Cross-Origin Resource Sharing, apply or prohibit requests from other domains.
- ngrok - Service for making our local server to public URL.

# Downloading and start
- Download project
- For launch require (command: npm i ), (npm install --save-dev typescript)
- Than start server (command: npm run build).
- For ngrok, need to write this (command: npx ngrok config add-authtoken 'your token') and than (command: npm run tunnel)
